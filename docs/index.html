cc-lark-channel

通过飞书操控 Claude Code，随时随地推进项目开发

1
安装并启动 Agent

在你的 Claude Code 所在机器上执行：

git clone https://github.com/totola147/cc-lark-channel.git
cd cc-lark-channel && npm install && npm run build
node dist/index.cjs --relay ws://RELAY_HOST:9000

首次启动会显示 6 位设备码，完成配对后执行：

node dist/index.cjs --install-service

注册为系统服务，开机自启、崩溃自动重启，无需手动管理。

2
飞书认证

点击下方按钮完成飞书 OAuth 登录：

飞书 OAuth 登录
3
输入设备码完成配对

✓
配对成功
