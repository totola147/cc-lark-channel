cc-lark-channel

通过飞书操控 Claude Code
随时随地推进项目开发

飞书发消息
→
Claude Code 执行
→
结果回传飞书

30 秒接入

  1. 克隆并构建 
    git clone https://github.com/totola147/cc-lark-channel.git
cd cc-lark-channel && npm install && npm run build
  2. 配对 
    node dist/index.cjs pair --relay wss://relay.cc-lark.dev
    获取 6 位配对码
  3. 飞书扫码 在飞书中添加 cc-lark-channel Bot，发送配对码

完成。现在可以在飞书里和 Claude Code 对话了。

接入方式

共享 Bot（推荐）
自有 Bot

零配置，无需创建飞书应用。使用公共 Bot + Relay 中转。

git clone https://github.com/totola147/cc-lark-channel.git
cd cc-lark-channel && npm install && npm run build
node dist/index.cjs pair --relay wss://relay.cc-lark.dev
# 飞书发送配对码后：
node dist/index.cjs --relay wss://relay.cc-lark.dev --token <token>

自建飞书应用，直连 WebSocket，数据不经过第三方。

git clone https://github.com/totola147/cc-lark-channel.git
cd cc-lark-channel && npm install && npm run build
# 配置 config.toml（填入 app_id/secret）
cp config.example.toml config.toml && vim config.toml
node dist/index.cjs --direct --config config.toml

飞书命令

命令说明
/cd <path>切换工作目录
/attach <id>接管 CLI 会话
/bg [name]当前任务放后台
/fg <id>切回前台
/sessions查看所有会话
/new新建会话
/stop停止生成
!<text>中断并执行新指令

For AI Agents

自动化安装（供 AI coding agent 执行）

# 检查环境
node --version  # >= 22
which claude    # Claude Code CLI 可用

# 克隆并构建
git clone https://github.com/totola147/cc-lark-channel.git ~/cc-lark-channel
cd ~/cc-lark-channel && npm install && npm run build

# 模式 A：共享 Bot
node dist/index.cjs pair --relay wss://relay.cc-lark.dev
# → 告诉用户 6 位码，让用户在飞书发送
# → 配对成功后：
node dist/index.cjs --relay wss://relay.cc-lark.dev --token <token>

# 模式 B：自有 Bot（用户提供 app_id/secret）
cp config.example.toml config.toml
# 编辑 config.toml 填入:
#   [lark] app_id / app_secret
#   [claude] cli_path = "$(which claude)", default_cwd = "$(pwd)"
node dist/index.cjs --direct --config config.toml

# 后台运行
pm2 start "node ~/cc-lark-channel/dist/index.cjs --direct --config ~/cc-lark-channel/config.toml" --name clc

# 验证
pm2 logs clc --lines 3
# 成功标志: "cc-lark-channel ready — listening for messages"

架构

共享 Bot 模式:
  飞书 → Cloud Relay → WebSocket tunnel → 本地 clc → Claude Code

自有 Bot 模式:
  飞书 → 本地 clc → Claude Code（直连）

代码和 API key 始终在用户本地，Relay 只转发消息。
